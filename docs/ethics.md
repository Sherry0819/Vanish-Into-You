# Ethics Statement

Vanish Into You does not attempt to recreate or replace human consciousness.

**Digital Profile** is a bounded representation derived from user-authorized textual traces:
- interpretability-first features (style/lexical/topic summaries)
- explicit uncertainty and source limits
- blocked sensitive requests (credentials, seed phrases, authentication)

The system should be used for memorialization and legitimate legacy transfer,
not impersonation or deceptive uses.
